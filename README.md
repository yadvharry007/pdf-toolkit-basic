# pdf-toolkit-basic
This app has basic html code for **splitting pdf**, **merging pdf** and **doc to pdf** 

### Split PDF
```
Input Data:
  - file: PDF file
  - split_type: 'individual' | 'every' | 'range'
  - every_n: number (if split_type='every')
  - page_range: string (if split_type='range', e.g., "1-5,7,10-12")
```

### Merge PDFs
```
Input Data:
  - files: Multiple PDF files
```
